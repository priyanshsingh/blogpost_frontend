Follow the steps given below to run the project:
1. Extract the zip file through any zip extractor like win-rar
2. Now open powershell or terminal server folder and run command "npm install" this will install dependencies required to run server side app.
3. After the step 2, run command "npm start" this will run the server side app at port 4000.
4. After the server side app is running successfully, goto client folder.
5. Open terminal/powershell in this folder and run command "npm install" this will install required dependencies to run the client side app.
6. After completion of step 5 run command "npm start" this will run the react app on port 3000, the app will open automatically after it is successfully compiled in browser.
7. Goto "http://localhost:3000/" and you will see the app running.
Note:
Make sure to run both client and server apps simultaneously
